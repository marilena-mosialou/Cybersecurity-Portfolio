# 🛠️ Tools Used in Email Analysis

This document outlines the tools used throughout the investigation process and their purpose.

---

## 🔍 Header Analysis

**Google Admin Toolbox (Messageheader)**
- Parses email headers
- Extracts SPF, DKIM, and DMARC results
- Identifies sending servers

---

## 🌐 Domain & IP Analysis

**WHOIS (whois.com)**
- Retrieves domain registration details
- Identifies domain ownership and registration date
- Helps detect recently created or suspicious domains

**AbuseIPDB**
- IP reputation analysis
- Identifies malicious activity history

---

## 🔗 URL Analysis

**URLScan.io**
- Observes URL behavior in a sandbox
- Detects redirects and final destinations

**VirusTotal**
- Aggregates results from multiple security vendors
- Identifies known malicious URLs or domains

---

## 🌍 Open Source Intelligence (OSINT)

**Google Search**
- Used to identify domain reputation and associations
- Helps validate whether infrastructure is linked to legitimate organizations

---

## 🧠 Key Insight

Tools support the investigation but do not determine the outcome.

Final conclusions are based on:
- Correlation of multiple data points
- Behavioral analysis
- Contextual understanding of the email
